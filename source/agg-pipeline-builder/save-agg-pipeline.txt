===============
Save a Pipeline
===============

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

You can save an aggregation pipeline so that you can find it later. If you
load a saved pipeline, you can edits changes to the pipeline stages without 
changing the original saved copy. You can also create a :ref:`view 
<view-details>` from your pipeline results.

Before You Begin 
----------------

To save a pipeline, you must first :ref:`create an aggregation pipeline 
<create-agg-pipeline>`.
